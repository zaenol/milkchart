# Syntax #

```
var chart = new MilkChart.Bar(table, [options])
```

# Arguments #

  1. table - (mixed) a table Element (or its id) that should be transformed
  1. options - (object, optional) a key/value set of options

# Options #

  * Same as [Column](Column.md)
